# Облачный провайдер виртуальных машин и контейнеров.

Проект - платформа для аренды виртуальных машин (на базе QEMU) и контейнеров (на базе Docker) с веб-интерфейсом на Streamlit.

Пользователь сначала регистрируется на платформе (если уже зарегистрирован, то входит с логином и паролем), затем, пользуясь интерфейсом, выбирает, что ему нужно создать - контейнер или ВМ и с какими параметрами. После чего может видеть список созданных им машин. Добавлять новые, удалять старые.

## Возможности

- Регистрация и авторизация пользователей
- Создание виртуальных машин на базе QEMU
- Создание контейнеров на базе Docker
- Управление машинами - запуск, остановка, удаление
- Простой мониторинг машин
- SSH-доступ к созданным машинам

## Стек технологий

### Backend
- **FastAPI** - REST API framework
- **SQLAlchemy** - ORM для работы с БД
- **SQLite** - база данных
- **Docker** - управление контейнерами
- **QEMU** - управление виртуальными машинами
- **Python** - основной язык проекта

### Frontend
- **Streamlit** - веб-интерфейс
- **Requests** - HTTP-клиент

## Требования к системе

- Python 3.8+
- Docker
- QEMU
- Пакеты python из requirements.txt
